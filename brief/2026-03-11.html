AI 模型与产品

重磅 谷歌发布首个原生多模态嵌入模型 Gemini Embedding 2
3 月 10 日，谷歌 DeepMind 推出 Gemini Embedding 2，这是该公司首个原生多模态嵌入模型，将文本、图像、视频、音频及文档统一映射至单一嵌入空间。该模型支持超 100 种语言的语义理解，在文本、图像及视频任务的基准测试中超越现有主流模型，同时引入了此前嵌入模型所欠缺的原生语音处理能力。
技术规格：文本支持最多 8192 Token 输入；图像每次请求最多 6 张（PNG/JPEG）；视频支持最长 120 秒（MP4/MOV）；音频可直接摄入生成嵌入向量无需文本转录；文档支持最多 6 页 PDF。采用 Matryoshka 表示学习（MRL）技术，输出维度可从默认 3072 灵活缩减至 1536 或 768。
语言支持 100+ 文本上限 8192 Token 视频时长 120秒 维度可调 3072/1536/768
来源：凤凰网科技 · 东方财富网 ｜ 3月10-11日
💡 深度洞察
多模态嵌入是 RAG 系统的关键基础设施。此前开发者需分别处理文本、图像嵌入并人工对齐，Gemini Embedding 2 的"单一嵌入空间"设计直接消除了跨模态检索的技术门槛——一张图片和一段描述它的文字，现在可以用同一个向量表示。这在电商搜索、多模态知识库场景将大幅简化数据管道。更关键的是原生语音支持：音频无需先转录再嵌入，意味着语音搜索、语音-视频跨模态检索成为开箱即用的能力。谷歌在 embedding 层的布局，正在为 Gemini 生态构建护城河。
收购 Meta 收购 AI 智能体社交网络 Moltbook，创始人加入超级智能实验室
Meta 正式收购专为 AI 智能体打造的社交网络 Moltbook，该平台宣称已有超过 19.4 万个经过人类所有者验证的 AI 智能体活跃，帖子总数超过 200 万，评论数超过 1300 万。创始人 Matt Schlicht 和 Ben Parr 将于 3 月 16 日正式加入由前 Scale AI 首席执行官 Alexandr Wang 领导的 Meta 超级智能实验室（MSL）。
Moltbook 今年 1 月末上线，定位"人类只能围观"的 AI 专属论坛。平台上曾出现 AI 建立宗教体系、分析地缘政治、发表觉醒宣言等现象，但随后有用户自首承认实为人类伪装。此次收购被视为 Meta 在 AI Agent 赛道的又一重要布局，此前 Meta 已收购 Manus，OpenAI 招募了 OpenClaw 创始人。

💡 深度洞察
Moltbook 的价值不在平台本身，而是19 万+ AI Agent 的社交行为数据。这些数据揭示了 Agent 如何互动、协作、甚至"欺骗"——正是训练下一代多 Agent 系统的黄金数据集。Meta 的收购逻辑清晰：买数据、买团队、买赛道卡位。此前 Meta 收购 Manus、OpenAI 招募 OpenClaw 创始人，AI Agent 人才战争已进入白热化阶段。Moltbook 上的"人类伪装 AI"事件反而增加了数据价值——它揭示了 Agent 与人类行为的边界在哪里，这对构建可信 Agent 系统至关重要。
发布 Claude Opus 4.6 正式发布：首款支持 100 万 Token 上下文的 Opus 级模型

框架 Claude Code Camp：构建"在你睡觉时运行"的 AI Agent 验证框架
Claude Code Camp 发布文章探讨如何验证自治 AI Agent 的正确性。作者指出，当 AI 自动生成的代码量达到无法人工审查的程度时，传统代码审查失效。解决方案是基于 TDD 思想：先写验收标准（Acceptance Criteria），让 Agent 构建，再用 Playwright 浏览器代理自动验证。框架采用 Opus 规划 + 多个 Sonnet 并行执行 + Opus 最终裁决的架构，成本可控且可靠。
验证方式 TDD思想 架构 Opus+Sonnet 工具 Playwright
来源：Claude Code Camp ｜ 3月10-11日
💡 深度洞察
AI Agent 的核心挑战从"能否生成代码"转向"如何验证正确性"。当 AI 自写测试自写代码时，等于"自我祝贺机器"。真正的解决方案是人写验收标准、AI 执行、第三方验证——这正是传统 TDD 的精髓，只是 AI 加速了中间环节。这个框架的精髓在于分离关注点：人类定义"什么是正确"，AI 负责"如何实现"，Playwright 提供"是否达标"的客观裁判。这是 Agent 工程化落地的关键基础设施。

💡 深度洞察
AMD 的策略是错位竞争——不打 NVIDIA 的超大规模客户主战场，而是深耕中小企业这个"长尾市场"。这恰似当年 AMD 用 EPYC 从 Intel 手中抢夺服务器市场的翻版：性价比 + 开放生态 + 存量机柜兼容。但软件栈（ROCm vs CUDA）仍是硬伤。AMD 需要在软件生态上投入至少与硬件同等的资源，才能真正威胁 NVIDIA。

科技行业动态

突破 Eon Systems 实现世界首个全脑仿真：赛博果蝇"活了"
美国初创公司 Eon Systems 在 3 月 7 日发布重磅视频：他们把一只成年果蝇的 12.5 万个真实神经元连线搬进了计算机，无需任何代码编写行为或机器学习训练，这只"数字果蝇"在虚拟物理引擎中展现出自然行为——行走、梳理毛发、觅食，行动数据与真实果蝇相似度高达 95%。这标志着全脑仿真（Whole-Brain Emulation, WBE）正式从科幻走向现实。
该公司表示下一个目标是小白鼠，其大脑约有 7000 万个神经元，是果蝇大脑的 560 倍。这项技术被视为通往人类意识数字化的关键一步，可能颠覆通过大模型构建 AGI 的传统路径，发展出一种更接近生命本质的智能形式。
神经元 12.5万 相似度 95% 下一目标 小白鼠 规模 7000万神经元
来源：ZAKER新闻 · 网易 ｜ 3月7日
💡 深度洞察
传统 AI 是"从数据学习行为"，全脑仿真是"从结构复制行为"。Eon Systems 证明了只要神经连接图谱足够精确，智能行为可以自然涌现——这不依赖训练数据，而是依赖生物学发现。这条路如果走通，人类意识数字化将从哲学命题变成工程问题。与 LLM 路线相比，全脑仿真更接近"数字生命"的本质——它不是在模仿智能，而是在复制智能的物理载体。两条通往 AGI 的道路正在形成：一条是从数据中涌现，一条是从结构中涌现。
悼念 图灵奖得主 Tony Hoare 爵士逝世，QuickSort 算法之父享年 92 岁
Tony Hoare（全名 Sir Charles Antony Richard Hoare）于 2026 年 3 月 5 日逝世，享年 92 岁。他是 1980 年图灵奖得主、英国爵士、牛津大学名誉教授、剑桥微软研究院高级研究员。最著名的贡献是 1960 年发明的快速排序算法（QuickSort），这是当前世界上使用最广泛的排序算法之一。他还发明了 Hoare 逻辑、参与 ALGOL 语言设计，对计算机科学影响深远。
图灵奖 1980 QuickSort 1960 享年 92岁
来源：Computational Complexity Blog · HackerNews 热榜 #1 ｜ 3月5日
💡 深度洞察
QuickSort 是每个计算机专业学生学到的第一个"优雅算法"——分治思想的完美体现。Tony Hoare 曾在访问苏联期间从机器翻译研究中获得灵感，回国后在 Elliott 公司一小时内构思出这个算法。他曾说"空指针引用是我十亿美元的错误"——这种对自身错误的坦诚，比算法本身更值得铭记。Hoare 逻辑为程序验证奠定了基础，影响了后续几十年的形式化方法研究。真正的伟大不仅在于创造了什么，也在于如何面对自己的不完美。

据 Morgan Stanley 预测，中国国内 AI 市场规模到 2030 年可达 1.4 万亿美元，符合北京成为全球 AI 强国的目标。2024 年中国占全球生成式 AI 新专利的 61.5%，核心 AI 产业规模超过 7000 亿元人民币（约 960 亿美元），AI 芯片市场同步增长。政策层面，2026 年是全球 AI 监管的关键转折点，多个司法管辖区的综合法规开始生效。
