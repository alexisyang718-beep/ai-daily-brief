模型与产品

AI编程 AMD 副总裁用 Claude Code 全程 AI 生成 GPU 驱动代码
AMD 企业副总裁 Anush Elangovan 发布了一个完全用 Python 编写的轻量级 Radeon 驱动测试工具。所有代码 100% 由 Claude Code 生成，他甚至一次都没打开过代码编辑器。该工具绕过 ROCm 软件栈，直接与 Linux 内核 GPU 接口交互，用于调试和硬件功能测试。项目展示了 AMD 内核接口的开放性，同时也证明 AI 编码工具已经能够独立完成专业级底层系统开发。
来源：搜狐科技/IT之家 ｜ 3月8日
发布 谷歌搜索向全美推出 Gemini Canvas AI 模式，"Google Zero"效应加剧
谷歌宣布在搜索中为所有美国用户推出 Gemini Canvas 功能。类似于 OpenAI 的 ChatGPT Canvas 和 Anthropic 的 Artifacts，将 Google 搜索引擎转变为由 Gemini 2.0 驱动的定制化聊天界面。用户可在搜索结果中直接与 AI 进行深度交互和内容创作，而不仅仅查看 AI 概述摘要。与此同时，谷歌"AI 概览"功能已导致多家科技媒体来自谷歌的流量暴跌 85%-97%，这一现象被称为"Google Zero"——AI 摘要直接回答用户问题，媒体网站获得的点击流量趋近于零。
来源：TechCrunch · TechDator · NPR · AI Magazine ｜ 3月7-8日
Agent Cursor 推出 Automations：7×24 小时 AI 自动化编码
Cursor 宣布推出 Cursor Automations，让 AI 7×24 小时自动审核、监控、修复代码并辅助研发流程。开发者只需配置智能体，即可持续监控并优化代码库，打造自动化"软件研发工厂"。所有自动化能力由云端 Agent 驱动，不消耗本地资源。标志着 AI 编程工具从"辅助编码"进入"自主编码"阶段。
来源：TechCrunch · Cursor 官网 ｜ 3月7-8日
发布 Claude Code v2.1.71 新增 /loop 命令：终端级周期任务自动化
Anthropic 发布 Claude Code v2.1.71，核心新增 /loop 命令——开发者可在终端内设定时间间隔，让 AI 自动周期性执行提示词或斜杠命令（如 /loop 5m check the deploy 每 5 分钟检查部署状态）。同步上线 cron 调度工具，支持标准 cron 语法在会话内配置复杂周期任务。此外还修复了长会话 stdin 冻结、语音模式启动卡顿等多项稳定性问题。
来源：Anthropic 官方 Changelog · Releasebot ｜ 3月7日

行业动向

财报 B 站首次全年盈利后加码 AI：Q4 AI 广告暴涨 180%，增量利润投入 AI
哔哩哔哩发布 2025 年 Q4 及全年财报：全年营收 303.5 亿元（+13%），调整后净利润 25.9 亿，首次实现全年盈利。AI 已成为 B 站最亮眼的增长引擎——
① AI 社区生态成型：每月超 1.2 亿用户在 B 站消费 AI 内容，Q4 AI 内容播放时长同比增长 53%，AI 辅助生产内容观看量较前年提升 100 倍。
② AI 广告爆发：Q4 AI 相关广告预算同比增长近 180%，全年增长超 150%。COO 李旎表示，B 站年轻用户科技属性强，AI 投放转化远超纯曝光。
③ 加码投入：CEO 陈睿定义 AI 本质为"提升产能和创作效率"，2026 年将推出 AI 视频翻译等辅助创作工具；CFO 樊欣确认将部分新增利润再投资 AI，聚焦内容理解、分发效率和 AIGC 创作工具。
这是中概互联网公司中首个明确将"全年盈利 → 增量利润 → AI 投入"形成正向闭环的案例，信号意义显著。
