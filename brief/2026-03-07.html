战略 Anthropic 推出 Marketplace：零佣金 AI 应用商店
3 月 6 日，Anthropic 正式推出 Anthropic Marketplace，为企业客户提供购买第三方软件的新平台。首批入驻服务包括 Snowflake、Harvey、Replit 等基于 Claude 模型构建的应用。核心策略：零佣金——不从交易中抽成，并允许客户将用于 Anthropic 自有服务的年度支出转移购买第三方工具，模式对标 AWS Marketplace 和 Azure Marketplace。此举标志着 Anthropic 正从模型公司向企业 AI 生态平台转型，尤其在对美国政府业务面临不确定性之际，加速拓展商业版图。
来源：Claude AI (X/Twitter) · Claude Marketplace 官网 · Bloomberg ｜ 3月6日
💡 深度洞察 ▼ 展开全文
Marketplace 标志着 Anthropic 从"模型供应商"向"AI 生态平台"跃迁。零佣金的本质不是不赚钱，而是借鉴 AWS 飞轮——每笔第三方交易都间接放大 Claude API 消耗量。模型公司做平台是 2026 年最重要的行业趋势之一，OpenAI 有 GPT Store、Google 有 Vertex AI Marketplace，AI 应用分发权之争正式开启。长期来看，AI 应用分发的终局不是"应用商店"，而是"Agent 调度中心"——用户不下载应用，而是直接让 Agent 跨应用完成任务。Marketplace 只是这个终局的过渡形态。对国内企业的启示：加速平台化转型，用免费分发层+付费算力层建生态。

一、对 Anthropic 自身：从"模型供应商"到"AI 生态平台"的关键跃迁

  • Anthropic 当前年化收入约 $140 亿，估值 $3800 亿（2026 年 2 月 Series G），但收入高度依赖模型 API 调用。Marketplace 的推出意味着 Anthropic 正从"卖算力"转向"卖生态"——通过平台化能力提升客户粘性和 LTV（客户终身价值）。
  • 零佣金 ≠ 不赚钱。真正的商业逻辑是：客户在 Marketplace 采购第三方工具时，底层消耗的仍是 Claude API 算力。每一笔第三方交易都间接放大了 Anthropic 的 Token 消费量。这与 AWS 早期"不靠 Marketplace 抽成赚钱，靠 EC2 消耗赚钱"的飞轮模型如出一辙。
  • 五角大楼事件后 Claude 用户激增（免费用户增长 >60%，日注册量翻 4 倍，付费用户翻倍），Marketplace 恰好承接住了这波流量红利，将消费端热度转化为企业端营收增量。

二、对行业：AI 应用分发权之争正式开启

  • 模型公司做平台，是 2026 年最重要的行业趋势之一。OpenAI 有 GPT Store（已整合到 ChatGPT），Google 有 Vertex AI Marketplace，现在 Anthropic 也入局。"谁拥有 AI 应用的分发权"正在成为比"谁的模型更强"更关键的竞争维度。
  • AWS Marketplace 2025 年第三方软件 GMV 已超 $150 亿，Azure Marketplace 体量相近。Anthropic 零佣金策略短期内会快速吸引 ISV（独立软件厂商）入驻，但能否形成足够交易密度是关键考验。
  • 首批入驻的 Snowflake（数据仓库）、Harvey（法律 AI）、Replit（编程 AI）代表了企业数据层→垂直行业→开发者工具的三角布局，Anthropic 瞄准的是 B2B 高客单价市场，而非 ChatGPT 式的 C 端应用分发。

三、对上下游的影响

  • 上游（云厂商）：Anthropic 与 AWS 的关系变得微妙——既是 AWS 的最大 AI 客户（亚马逊累计投资 $80 亿），又在应用层与 AWS Marketplace 直接竞争。如果 Anthropic 能将企业客户的采购预算从 AWS 抢走，Amazon 的 AI 变现逻辑将受到冲击。
  • 下游（AI 应用开发者）：零佣金 + 允许客户复用 Anthropic 年度预算 = 降低了 AI 应用的获客成本。对于基于 Claude 构建的 SaaS 公司而言，这是一个明确的利好信号。
  • 竞争对手：OpenAI 的 GPT Store 至今未公布收入数据，Google 的 Vertex 市场份额有限。Anthropic 的零佣金策略会逼迫竞争对手跟进降费，整个行业的应用分发层将进入价格战。

四、对国内企业的启示

  • 国内大模型公司应加速平台化转型。目前百度文心、阿里通义、月之暗面的商业模式仍停留在"模型 API + 应用"，尚无一家推出类 Marketplace 的第三方应用分发平台。随着模型能力趋同，"生态厚度"将成为决胜关键。
  • Anthropic 的"零佣金"策略值得研究：以短期让利换取平台规模效应，本质是一场"用户注意力×API 消耗量"的飞轮赌注。国内企业可借鉴此策略，用免费分发层 + 付费算力层的模式快速建立应用生态。
  • 五角大楼事件证明：在 AI 时代，"价值观品牌"可以直接转化为商业增长。对于国内企业而言，数据安全和用户隐私同样可以成为差异化竞争的核心叙事。

五、未来趋势判断

  • 2026 年下半年，"AI Marketplace"将成为行业标配。预计 Meta（Llama 生态）、Mistral（欧洲市场）都将推出类似平台。
  • 长期来看，AI 应用分发的终局不是"应用商店"，而是"Agent 调度中心"——用户不下载应用，而是直接让 Agent 跨应用完成任务。Marketplace 只是这个终局的过渡形态。
  • Anthropic 的 $3800 亿估值已隐含了"平台公司"预期。如果 Marketplace 能在 12 个月内达到 $10 亿+ GMV，将验证"模型公司→平台公司"的转型逻辑，并可能推动其 IPO 进程。

国内首个手机厂商级系统 AI Agent。与传统语音助手的本质区别：miclaw 是"执行引擎"而非"对话工具"，封装 50+ 系统级工具，具备推理-执行闭环。依托小米 IoT 8 亿+ 设备生态，可调度手机、汽车、家电形成跨设备任务链。端侧 Agent 对芯片的推理算力和能效比提出了更高要求，高通/联发科/小米自研澎湃芯片将面临"端侧推理性能"的新军备竞赛。模型层面，端侧部署要求模型足够小且快，会加速小模型蒸馏和MoE 架构在手机端的落地。2026 年将成"手机 Agent 元年"，小米率先出手意味着中国厂商首次在移动 AI 赛道站在起跑线前面。

一、对小米自身：从"硬件+互联网"到"硬件+AI 原生"的战略升级

  • 这是国内首个手机厂商级的系统级 AI Agent 产品。与传统语音助手（Siri、小爱同学）的本质区别在于：miclaw 不是"对话工具"，而是"执行引擎"——它以系统应用身份运行，封装 50+ 系统级工具，具备推理-执行闭环能力（analyze → select tool → execute → review → iterate）。
  • 小米将 miclaw 定位为"人车家全生态"的统一 AI 入口。依托小米 IoT 平台连接超 8 亿台设备的生态优势，miclaw 的想象空间远大于单一手机 Agent：它可以调度手机、汽车、家电形成跨设备任务链。这是苹果 Siri、Google Assistant 至今未实现的能力层级。
  • 自研 MiMo 大模型是关键。小米未采用第三方模型（如通义/文心），而是完全基于自研模型构建端侧推理能力，并支持 MCP（Model Context Protocol）接入第三方 SDK。这意味着小米对 AI 能力的掌控度和迭代速度远高于依赖外部模型的厂商。
  • 五年 2000 亿研发投入承诺（覆盖芯片、AI、机器人），加上小米 17 系列硬件作为首发平台，说明小米正将 AI 视为与硬件同级的核心战略。

二、对行业：手机端 Agent 赛道正式从概念走向产品

  • 时间窗口意义重大。miclaw 的发布紧跟 OpenClaw 全球爆发（Discord 11.9 万用户、腾讯大厦排长队部署），精准卡位了"AI Agent 从 PC 端向移动端迁移"的行业拐点。小米将 Agent 的落地战场从浏览器/桌面拉到了手机终端，这是一个全新的竞争维度。
  • 手机 Agent 的核心壁垒在于系统级权限。相比第三方 Agent 应用（如 Perplexity、Rabbit R1），手机厂商天然拥有操作系统层的完整控制权——可以读取短信、日历、通知、传感器数据，可以调用系统 API 执行操作。这使得手机厂商在 Agent 赛道拥有其他玩家无法复制的结构性优势。
  • 支持 MCP 协议是前瞻性布局。MCP（由 Anthropic 提出）正在成为 AI Agent 的"USB 接口标准"，小米率先支持意味着第三方开发者可以低成本接入 miclaw 生态，有望催生手机端的"Agent 应用市场"。

三、对上下游的影响

  • 上游（芯片/模型）：端侧 Agent 对芯片的推理算力和能效比提出了更高要求。高通/联发科/小米自研澎湃芯片将面临"端侧推理性能"的新军备竞赛。模型层面，端侧部署要求模型足够小且快，会加速小模型蒸馏和MoE 架构在手机端的落地。
  • 下游（应用开发者）：传统 App 开发模式可能被重构。如果 Agent 能直接操控第三方应用完成任务，用户对"打开 App → 手动操作"的需求将下降。对于工具类/效率类 App 而言，这既是机会（被 Agent 调用 = 获得分发）也是威胁（用户不再直接使用 UI）。
  • 同行竞争：华为（盘古）、OPPO/vivo（蓝心/蓝河）、三星（Galaxy AI）都在布局手机 AI，但尚无类似"系统级 Agent"产品落地。小米率先封测可能抢占 3-6 个月的先发窗口。

四、对国内企业的启示

  • 手机厂商做 AI Agent 的逻辑是成立的。手机是最高频的计算设备（日均使用 4+ 小时），拥有最丰富的个人上下文数据（位置、通讯、消费、健康），天然适合 Agent 落地。国内手机厂商应抓住这一窗口期。
  • 小米的"三级记忆管理"机制（关键决策留存、冗余交互压缩、核心指令本地优化）使对话效率提升 30%、存储开销降低 45%，展示了端侧 Agent 在资源受限环境下的工程可行性。这一方案值得其他厂商借鉴。
  • "封测 + 邀请制 + 实验性产品"的发布策略值得学习。小米主动降低用户预期（明确说不稳定、建议勿在主力机安装），既控制了口碑风险，又制造了稀缺感和话题度。

五、未来趋势判断

  • 2026 年将是"手机 Agent 元年"。下半年苹果 WWDC 预计将推出系统级 Siri Agent，Google I/O 也将更新 Android Agent 能力。小米率先出手意味着中国厂商在这一轮竞争中首次站在了起跑线前面。
  • 端侧 Agent 将催生新的"注意力经济"：谁的 Agent 帮用户完成的任务最多，谁就掌握了新的流量分发权。传统 App 的"搜索→下载→打开"模式可能被"一句话→Agent 执行"模式逐步替代。
  • 小米"人车家全生态 + Agent 执行引擎"的组合如果跑通，将从"智能设备公司"升级为"AI 服务公司"，商业模型从硬件毛利转向服务收入（Agent 订阅、AI 增值服务、生态分发佣金），长期想象空间巨大。

英伟达从"卖 FLOPS"转向"卖 Token 经济学"，数据中心单季营收 $623 亿，首次全年不更新消费级 GPU。Blackwell 推理成本降 10x 的核心数据：每百万 Token 的硬件摊销成本从 $0.20 降至 $0.05 以下。Blackwell 上运行开源模型的评估速度提升 3-5 倍，这意味着过去需要一周完成的模型评估，现在可以在 1-2 天内完成。未来 AI 推理成本将再降 10-100 倍。对国内启示：国产推理芯片达 Hopper 级即可抢份额，液冷产业链是不受管制的确定性机会。

一、英伟达的业务战略变化：从"卖 GPU"到"卖推理经济学"

  • 数据中心业务已彻底压倒游戏业务。FY2026 Q4（截至 2026 年 1 月），英伟达数据中心营收达 $623 亿（单季），而游戏业务收入占比已降至历史低位。更具标志性的是：英伟达宣布 2026 年不发布任何新消费级游戏 GPU——这是自 1990 年代以来首次，资源全面倾斜 AI 数据中心。
  • 英伟达正在重新定义自己的价值主张。过去卖的是"FLOPS"（算力），现在卖的是"Tokenomics"（Token 经济学）。GTC 2026 的核心叙事不是"Blackwell 有多快"，而是"每美元能产出多少 Token"。这标志着英伟达从硬件性能竞赛转向推理成本效率竞赛。
  • 传闻英伟达正在洽谈收购 Groq（LPU 推理芯片公司），这是一个防御性战略：当 Google TPU、AMD MI355X、Groq LPU 在推理市场蚕食份额时，英伟达选择"买下潜在颠覆者"来巩固推理领域的统治地位。

二、"推理成本降 10 倍"的实际影响——关键数据解读

  • 核心数据：据 VentureBeat 报道及英伟达官方数据，Blackwell（GB200 NVL72）相比 Hopper（H100），在配合优化软件栈和开源模型时，Token 单价降低 4x-10x。具体案例：
DeepInfra $0.20→$0.05/百万Token Sully.ai(医疗) 成本降90% Decagon(语音客服) 成本降6x
  • 采购成本变化：H100 单卡市价约 $25,000-30,000，B200 单卡预估 $30,000-40,000（涨价 ~30%），但每卡产出的推理 Token 数提升 5-10 倍。折算后，每百万 Token 的硬件摊销成本从 $0.20 降至 $0.05 以下。对于一个日均处理 10 亿 Token 的中型 AI 公司，年化硬件成本可从 ~$7300 万降至 ~$1800 万，节省超 $5000 万。
  • Token 降价的传导效应：推理成本降 10 倍直接推动 API 定价下降。2024 年 GPT-4 级模型的 API 价格为 ~$30/百万 Token，到 2026 年初已降至 ~$3/百万 Token。Blackwell 大规模部署后，预计推动行业价格进一步降至 $0.3-1.0/百万 Token。这意味着：
2024年 GPT-4 级 ~$30/M Token 2026年初 ~$3/M Token Blackwell 后 $0.3-1.0/M Token
  • 对模型迭代速度的影响：推理成本下降 → 模型评估成本下降 → 研发团队可以跑更多实验 → 模型迭代周期缩短。据 Baseten 和 Together AI 的实际数据，Blackwell 上运行开源模型（如 Llama 3、DeepSeek-R1）的评估速度提升 3-5 倍，单个 DGX 系统可达 30,000+ tokens/s 吞吐量。这意味着过去需要一周完成的模型评估，现在可以在 1-2 天内完成。
  • 关键技术因素：10 倍降幅不仅来自硬件升级（仅贡献 ~2 倍），而是硬件+软件+模型架构三层协同的结果。具体包括：① Blackwell 原生 NVFP4 低精度格式（将 FP8 精度减半，贡献额外 2 倍）；② MoE（混合专家）模型在 NVLink 互联下的高效通信；③ TensorRT-LLM + Dynamo 推理引擎的软件优化。

三、芯片技术迭代启示

  • 从"训练为王"到"推理为王"的范式切换。H100 时代的核心卖点是训练性能（FP8 3,958 TFLOPS），Blackwell 的核心卖点变成了推理效率（FP4 20,000 TFLOPS，推理吞吐提升 30 倍）。这反映了行业现实：大模型训练需求增速放缓（头部公司已建成大规模训练集群），而推理需求随 AI 应用普及呈指数级增长。
  • 低精度计算是未来 3 年芯片竞争的核心战场。从 FP16（A100）→ FP8（H100）→ FP4（Blackwell），每一代精度减半都带来吞吐翻倍。下一代 Rubin 架构（2027 年）预计将支持 FP4/INT4 混合精度，目标在 Blackwell 基础上再降 10 倍推理成本。
  • 液冷成为刚需。GB200 NVL72 采用液冷散热，功耗 ~72kW/机柜，性能密度是 H100 风冷方案的 ~4 倍。这意味着新建数据中心必须配套液冷基础设施，对整个数据中心供应链（制冷系统、CDU、管路）形成拉动。

四、对 AI 行业的影响

  • "AI 推理平价化"加速到来。当推理成本降到 $1/百万 Token 以下，大量过去"算不起账"的 AI 应用场景将变得经济可行：实时 AI 客服、个性化内容生成、端侧 Agent 持续运行等。AI 的商业化拐点将从"能不能做"变成"值不值得做"。
  • 开源模型 + Blackwell = 对闭源模型的生存威胁。VentureBeat 数据显示，Sully.ai 使用开源模型在 Blackwell 上的推理成本比闭源模型降低 10 倍，且响应速度提升 65%。当开源模型的性价比碾压闭源 API，OpenAI/Anthropic/Google 的 API 定价权将被严重削弱。
  • 竞争格局：英伟达"推理帝国"面临三面夹击。① Google TPU 已获 Anthropic 大额订单，凭能效优势切入商用市场；② AMD MI355X 在中端推理市场提供价格替代；③ Groq LPU 在超低延迟场景（语音 AI）具备颠覆性优势。英伟达虽仍占推理市场 >80% 份额，但护城河正在收窄。

五、对国内企业的启示

  • 国产芯片的机遇窗口。Blackwell 受美国出口管制限制，中国企业无法直接采购。但推理成本"每年降 10 倍"的趋势是全球性的，国产推理芯片（如华为昇腾 910C、寒武纪 MLU590、海光 DCU）只要能达到 Hopper 级别的推理效率，在中国市场就能获得可观份额——因为中国客户的替代选项非常有限。
  • Token 降价将加速中国 AI 应用落地。当前国内大模型 API 价格已处于"百万 Token 一块钱"的卷价阶段（DeepSeek-V3 的 API 价格约 $0.27/百万 Token），Blackwell 带来的全球性成本下降将进一步压缩定价空间，但也意味着更多中小企业"用得起 AI"。
  • 液冷产业链是确定性机会。英伟达 Blackwell 全面转向液冷，中国是全球最大的数据中心市场之一，液冷相关的 CDU（冷量分配单元）、冷板、管路、冷却液供应商将直接受益。这是一条不受出口管制限制的供应链机会。

六、未来趋势判断

  • 2026-2027 年，AI 推理成本将再降 10-100 倍。英伟达 Rubin 架构（2027）目标是在 Blackwell 基础上再降 10 倍；叠加算法优化（MoE、推测解码、KV Cache 压缩），到 2028 年 GPT-4 级推理成本可能降至 $0.01/百万 Token——届时 AI 将真正成为"零边际成本"的基础设施。
  • "推理即服务"（Inference-as-a-Service）将成为比"模型即服务"更大的市场。Baseten、Together AI、Fireworks AI 等推理服务商正在快速崛起，预计 2027 年全球 AI 推理服务市场规模将超 $500 亿。
  • 英伟达放弃消费级 GPU 更新，短期看是"聚焦高利润业务"，长期看可能留下战略隐患——如果 AI PC 和端侧推理需求爆发，缺乏消费级产品线的英伟达可能将这一市场拱手让给 AMD、Intel 和高通。
