🔥 微博独家 阿里发布全球首个企业级 Agent 平台"悟空"：打通钉钉 2000 万企业，底层 CLI 化
3 月 17 日，阿里巴巴发布全球首个企业级 AI 原生工作平台——"悟空"，即日起邀测，直接内置到超 2000 万企业组织的钉钉中。核心：8 亿用户钉钉重写底层代码，全面 CLI 化改造，让 Agent 原生操作上千项能力，"把钉钉打碎重建"——底层从 GUI 改为 CLI 让 Agent 直接操作。CEO 吴泳铭亲自挂帅新成立 ATH（Alibaba Token Hub）事业群，涵盖通义实验室、千问、悟空事业部，定位"创造/输送/应用 Token"。同日通义开源影视级配音模型 Fun-CineForge。

💡 深度洞察
悟空核心创新不在模型，在"CLI 化改造"——把钉钉从人操作的 GUI 改造为 Agent 可直接调用的命令行接口。这是 Agent 落地企业的关键范式：GUI 为人设计，CLI 才为 Agent 设计。阿里正在将 Agent 从"个人玩具"推向"企业生产力工具"。
🔥 微博独家 百度秒哒发布全球首个 OpenClaw 应用开发 Skill："对话即应用"
3 月 16 日，百度秒哒推出全球首个 OpenClaw 应用开发 Skill——"Miaoda App Builder"。业内首次将专业应用开发能力封装为智能体可调用的技能包。用户配置 API Key 后，一句话即可让"龙虾"完成专业网站开发——从前端到后端、代码生成到部署上线全自动，支持网页、小程序、游戏等应用，真正实现"对话即应用"。

💡 深度洞察
秒哒 Skill 打通了"自然语言 → 开发 → 部署上线"全链路。OpenClaw "龙虾"不再只是聊天机器人，而是能直接产出可用产品的生产工具。每个通过秒哒开发的应用背后都在消耗百度 API 调用量——这是百度模型能力变现的重要路径。

mini/nano 的策略：用极致性价比抢占 Agent 运行时市场。"龙虾"每个任务需数百至数千轮 API 调用，mini 用 1/3 价格逼近旗舰性能。nano 瞄准"Agent 子任务"——数据分类、格式提取不需满血推理但需极低延迟。OpenAI 正在为 Agent 生态构建分层模型矩阵。
热点 Anthropic 推出 Claude Dispatch：手机派发任务，桌面端 AI 持续执行
Anthropic 发布 Claude Cowork 新功能 Dispatch（研究预览版）。核心理念：一个持续运行的 Claude 对话驻留在桌面端，用户可随时从手机端 Claude App 向其发送任务指令——无需终端、无需发起 session，直接消息派发即可。这与 Claude Code 的 /remote-control 方向一致，但面向非开发者更易用。发布推文获 167 万次浏览和近 9000 个点赞，成为今日 AI 社区最热话题。

💡 深度洞察
Dispatch 标志着 AI 助手从"被动回应"向"异步调度"转型。传统对话模式要求用户实时在线，而 Dispatch 允许用户像给下属发微信一样布置任务、随后查看结果。它让 Claude 拥有了类似"常驻进程"的存在感——这与 OpenAI Subagent 的理念异曲同工：未来 Agent 不是"一次对话"，而是"持续运行的进程"。

芯片与算力

🔥 GTC 2026 NVIDIA GTC 全面发布：Vera Rubin + 七芯片集成 Groq + Feynman 提前曝光
GTC 2026 大会（3/16-17，圣何塞）黄仁勋发布多项重磅：
① Vera Rubin NVL144：Blackwell 继任者，2026H2 量产，3nm EUV + HBM4，单卡 288GB，FP4 3.6 PFLOPS（前代 1.1），推理成本降 10 倍。
② 七芯片集成 Groq：整合 170 亿美元收购，首 Token 延迟 <0.1ms，专为大规模 Agent 优化。
③ NemoClaw：为 OpenClaw "龙虾"设计的一键部署软件栈。
④ Feynman 架构：下一代 GPU，台积电 1.6nm + 硅光子光互连，带宽↑10 倍、能耗↓90%，推理 = Blackwell 5 倍，2028 量产。
黄仁勋预测 2027 年 AI 芯片营收目标 1 万亿美元。
Vera Rubin FP4 3.6 PFLOPS Groq 首 Token <0.1ms 2027 营收 $1T
来源：C114 ｜ 新浪 ｜ 多源/GTC ｜ 2026-03-17
💡 深度洞察
Vera Rubin 推理成本降 10 倍直接改变 Agent 经济学——当前"养龙虾"最大成本是 API 推理费。Groq 整合补齐推理短板，0.1ms 首 Token 让 AI 对话接近即时。Feynman 硅光互连是面向 2028 的赌注——传统电信号互连已成算力瓶颈，光互连是唯一出路。
重磅 黄仁勋放风重返中国市场：H200 重启生产，但"零收入"困局未解
据彭博社/观察者网 3 月 18 日报道，黄仁勋在 GTC 大会上亲口确认：NVIDIA 已为多家中国客户拿到 H200 出口许可，已收到采购订单并正式重启生产。"我们的供应链正在全力运转起来，与两三周前的情况截然不同。"他还补充说，"特朗普总统的意图是美国应保持领先，同时不轻易放弃相关市场。"

但附带条件苛刻：25% 销售收入上缴美国政府，对华出货量不得超过美国本土的 50%。中国曾贡献英伟达四分之一营收，如今占比已大幅缩水。黄仁勋此前表示中国 AI 芯片市场规模可达 500 亿美元，但在 25% 上缴 + 合规成本叠加下，实际利润极薄——业内称之为"零收入"困局。
上缴比例 25% 出货上限 美国本土 50% 中国市场 ~$500 亿
来源：观察者网/彭博社 ｜ 腾讯网 ｜ 央视网 ｜ 2026-03-18
💡 深度洞察
黄仁勋"放风"意味深长：从"零订单"到"供应链全力运转"，逆转叙事精准踩在 GTC 大会时间窗口，意在提振投资者信心。但现实是：25% 上缴 + 50% 出货上限 + 合规审批成本叠加后，H200 对华销售利润极薄。真正的变数不在 H200，而在 Groq 推理芯片能否以"非管制品类"身份合规进入中国——这才是 NVIDIA 重返中国市场的关键棋子。
新品 英伟达 Groq 3 LPU 推理芯片正式发布：三星 4nm 代工，中国市场成关键变量
黄仁勋在 GTC 主题演讲中正式发布 Groq 3 LPU（Language Processing Unit，语言处理单元）推理芯片，纳入 Vera Rubin AI 平台。这是英伟达去年底以 170 亿美元收购 Groq 后首次将 LPU 以量产产品推向市场。三星 4nm 工艺制造，基于 Groq 3 的 LPX 机架预计今年 H2 面世。

天风国际分析师郭明錤预测：2026-2027 年 LPU 总出货量 400-500 万颗，2026 年机柜出货 300-500 个，2027 年增至 15,000-20,000 个。据路透社知情人士透露，英伟达正准备推出面向中国市场的 Groq 芯片——非降级版、非"特供版"，定位 AI 推理，预计 Q2 上市。由于 LPU 是推理专用芯片而非训练 GPU，在出口管制框架中可能不受 H200 同等限制。
工艺 三星 4nm 出货 400-500 万颗 中国 Q2 上市
来源：腾讯网/郭明錤 ｜ 搜狐/GTC ｜ 路透社 ｜ 2026-03-17~18
💡 深度洞察
Groq LPU 是 NVIDIA 重返中国市场的"合规后门"——推理芯片不直接用于模型训练，在美国出口管制分类中可能绕开 H200 所受的严格限制。如果 Q2 顺利在华上市，将填补中国对低延迟推理芯片的巨大需求空白。郭明錤 400-500 万颗的出货预测说明市场已在为"推理时代"的到来下注。三星代工而非台积电，则是供应链多元化的信号。
🔥 微博独家 SK 集团董事长 GTC 预警：内存芯片短缺将持续至 2030 年
3 月 17 日，SK 集团董事长崔泰源在 GTC 表示：全球内存芯片短缺可能持续到 2030 年。AI 对 HBM 的超大需求造成晶圆短缺，增加产能至少需 4-5 年。AI 存储供需缺口超 30%，DRAM/NAND/HBM 价格将持续上涨。SK 海力士 CEO 即将公布稳定 DRAM 价格新计划，同时评估赴美上市。
供需缺口 >30% 产能周期 4-5 年 短缺至 2030
来源：新浪财经 ｜ IT之家 ｜ @CNMO @SK集团（微博）｜ 2026-03-17
💡 深度洞察
SK 集团是全球最大 HBM 供应商之一，"短缺至 2030"意味着今天的手机涨价只是开始——AI 对存储的虹吸效应将在未来 4 年持续推高所有电子产品成本。这也解释了 Vera Rubin 切换 HBM4 的必要性：必须靠技术升级提效，因为产能不可能跟上需求。
